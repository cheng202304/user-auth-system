## 系统简介

用户认证系统是一个提供完整用户注册、登录、管理功能的安全认证系统。系统采用现代化的前后端分离架构，支持多角色权限管理。

### 系统功能

- 用户注册（自动生成6位数字账号）
- 用户登录（支持记住密码）
- 密码修改
- 个人资料管理
- 用户管理（管理员功能）
- 角色权限控制

## 快速开始

### 1. 启动系统

**方式一：一键启动（推荐）**

```bash
# Linux/Mac
./start.sh

# Windows
start.bat
```

**方式二：手动启动**

```bash
# 安装依赖
npm run install:all

# 初始化数据库
cd backend
npm run migrate
npm run seed

# 启动后端
npm run dev:backend

# 启动前端（新终端）
cd ../frontend
npm run dev:frontend
```

### 2. 访问系统

启动成功后，打开浏览器访问：

- 前端地址：http://localhost:3001
- 后端API：http://localhost:3000

### 3. 登录系统

默认超级管理员账号：

- 账号：`000000`
- 密码：`123456`

> **注意**：首次登录后请立即修改密码！

## 功能操作指南

### 游客操作

#### 1. 浏览首页

- 访问 http://localhost:3001 即可查看首页
- 首页展示系统介绍和功能说明

#### 2. 用户注册

1. 点击导航栏「注册」按钮
2. 系统自动生成6位数字账号
3. 注册成功后记录账号和初始密码 `123456`
4. 使用账号密码登录

#### 3. 用户登录

1. 点击导航栏「登录」按钮
2. 输入账号和密码
3. 可勾选「记住密码」
4. 点击登录按钮

### 已登录用户操作

#### 1. 查看个人资料

1. 登录成功后，点击导航栏「个人中心」
2. 查看个人账号、用户名、邮箱、手机号等信息

#### 2. 修改个人资料

1. 进入个人中心页面
2. 点击「编辑资料」
3. 修改用户名、邮箱、手机号等信息
4. 点击「保存」按钮

#### 3. 修改密码

1. 进入个人中心页面
2. 点击「修改密码」
3. 输入当前密码
4. 输入新密码（至少6位）
5. 确认新密码
6. 点击「确认修改」
7. 系统提示修改成功，需要重新登录

#### 4. 退出登录

1. 点击导航栏「退出」按钮
2. 系统清除登录状态
3. 页面刷新，恢复未登录状态

### 管理员操作

#### 1. 用户查询

1. 登录管理员账号
2. 进入「用户管理」页面
3. 可按以下条件筛选：
   - 账号（精确查询）
   - 用户名（模糊查询）
   - 角色（管理员/教师/学生）
   - 状态（正常/禁用）
4. 支持分页查看，每页20条

#### 2. 新增用户

1. 进入「用户管理」页面
2. 点击「新增用户」按钮
3. 填写用户信息：
   - 用户名（必填）
   - 邮箱（可选）
   - 手机号（可选）
   - 角色（教师/学生）
4. 系统自动分配6位账号
5. 默认密码为 `123456`

#### 3. 修改用户

1. 进入「用户管理」页面
2. 找到目标用户，点击「编辑」按钮
3. 可修改：
   - 用户名
   - 邮箱
   - 手机号
   - 角色
   - 账号状态（正常/禁用）
4. 点击「保存」

#### 4. 重置密码

1. 进入「用户管理」页面
2. 找到目标用户，点击「重置密码」
3. 确认后，用户密码将被重置为 `123456`

#### 5. 删除用户

1. 进入「用户管理」页面
2. 找到目标用户，点击「删除」
3. 确认删除操作
4. 用户被删除，账号释放可重新分配

> **注意**：超级管理员账号（000000）不能被修改或删除

## 常见问题

### 1. 忘记密码怎么办？

请联系系统管理员重置密码。

### 2. 账号被锁定怎么办？

登录失败5次后，账号将被锁定30分钟。请30分钟后再尝试登录，或联系管理员解锁。

### 3. 如何注册新账号？

点击首页「注册」按钮，系统会自动分配一个6位数字账号。初始密码为 `123456`。

### 4. 为什么无法登录？

- 检查账号和密码是否正确
- 检查账号是否被锁定
- 检查账号是否被禁用
- 联系管理员确认账号状态

### 5. 如何修改个人头像？

在个人中心页面，点击头像区域进行上传。支持 JPG、PNG 格式，大小限制 2MB。

### 6. 管理员可以删除自己的账号吗？

管理员可以删除其他管理员账号，但不能删除自己。超级管理员账号（000000）不能被任何方式修改或删除。

### 7. 技术支持

如遇到无法解决的问题，请联系系统管理员。
