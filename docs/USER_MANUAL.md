## 系统概述

用户认证系统是一个安全可靠的用户注册登录平台，支持多角色管理和完整的用户生命周期管理。系统提供简洁商业风格的用户界面，适用于企业、教育机构等场景。

### 主要功能
- **用户注册**: 自动生成6位数字账号
- **用户登录**: 账号密码认证，支持记住密码
- **个人中心**: 个人信息管理、密码修改
- **用户管理**: 管理员可管理所有用户
- **多角色支持**: 超级管理员、管理员、教师、学生
- **安全保护**: 登录失败锁定、密码加密存储

## 快速开始

### 1. 访问系统
打开浏览器，访问系统地址：
- **开发环境**: http://localhost:3001
- **生产环境**: https://your-domain.com

### 2. 注册账户
1. 点击首页的"**注册**"按钮
2. 填写用户名、邮箱和密码
3. 点击"**注册**"按钮
4. 系统自动生成6位数字账号
5. 默认密码为 `123456`，首次登录后必须修改

### 3. 登录系统
1. 在首页点击"**登录**"按钮
2. 输入您的6位数字账号
3. 输入您的密码
4. 点击"**登录**"按钮
5. 登录成功后自动跳转到仪表盘

## 用户角色说明

系统支持四种用户角色，每种角色有不同的权限：

| 角色 | 权限说明 | 账号示例 |
|------|----------|----------|
| **超级管理员** | 拥有所有权限，可管理所有用户 | 000000 |
| **管理员** | 可管理普通用户，不能管理其他管理员 | 123456 |
| **教师** | 可管理自己的信息和相关功能 | 123457 |
| **学生** | 可管理自己的信息 | 123458 |

> **注意**: 超级管理员账号固定为 `000000`，不能被删除或禁用。

## 首页使用

### 首页布局
首页分为三个主要区域：

1. **顶部导航栏**
   - 左侧：系统名称
   - 右侧：根据登录状态显示不同按钮

2. **主体内容区**
   - 欢迎标语
   - 系统功能介绍
   - 操作按钮

3. **底部区域**
   - 版权信息
   - 联系方式

### 导航栏状态

#### 未登录状态
- "**登录**"按钮：跳转到登录页面
- "**注册**"按钮：跳转到注册页面

#### 已登录状态
- "**个人中心**"链接：跳转到个人中心页面
- "**退出**"按钮：执行登出操作

## 账户管理

### 用户注册
1. 访问首页，点击"**注册**"按钮
2. 填写以下信息：
   - **用户名**: 2-20个字符
   - **邮箱**: 有效的邮箱地址
   - **密码**: 至少6位字符
3. 点击"**注册**"按钮
4. 系统会自动生成6位数字账号
5. 注册成功后，使用生成的账号和默认密码 `123456` 登录

> **重要**: 新用户首次登录后必须修改默认密码。

### 用户登录
1. 访问首页，点击"**登录**"按钮
2. 输入您的6位数字账号
3. 输入您的密码
4. 点击"**登录**"按钮

#### 登录安全机制
- **密码错误**: 连续5次错误后，账号锁定30分钟
- **记住密码**: 可选择记住登录状态
- **会话管理**: 登录状态在浏览器关闭后保持7天

### 用户登出
1. 在任何页面，点击导航栏的"**退出**"按钮
2. 系统会清除登录状态
3. 自动跳转回首页

## 个人中心

登录后，点击导航栏的"**个人中心**"链接进入个人中心页面。

### 个人信息管理
在"**个人信息**"标签页中，您可以：

1. **修改用户名**
   - 用户名长度必须在2-20个字符之间
   - 不能为空

2. **修改邮箱**
   - 必须是有效的邮箱格式
   - 用于接收系统通知

3. **修改手机号**
   - 必须是11位中国手机号码
   - 格式：1[3-9]开头的11位数字

4. **上传头像**
   - 支持 JPG、PNG 格式
   - 文件大小限制：2MB以内
   - 输入头像的URL地址

5. **查看只读信息**
   - **账号**: 您的6位数字账号（不可修改）
   - **角色**: 您的用户角色（不可修改）

### 修改密码
在"**修改密码**"标签页中，您可以：

1. **输入当前密码**
   - 必须输入正确的当前密码

2. **输入新密码**
   - 长度至少6位
   - 不能与当前密码相同

3. **确认新密码**
   - 必须与新密码一致

4. **保存更改**
   - 修改成功后需要重新登录

### 操作提示
- **保存成功**: 显示绿色成功消息
- **验证失败**: 显示红色错误消息，说明具体原因
- **必填字段**: 带有红色星号(*)的字段为必填项

## 管理员功能

### 用户管理（超级管理员/管理员）
管理员可以访问"**用户管理**"功能，进行以下操作：

#### 查看用户列表
- **分页显示**: 每页显示20条记录
- **排序**: 按账号升序排列
- **列信息**: 账号、用户名、邮箱、角色、状态、注册时间

#### 筛选用户
支持三种筛选方式：

1. **按角色筛选**
   - 超级管理员
   - 管理员  
   - 教师
   - 学生

2. **按状态筛选**
   - 正常
   - 已禁用

3. **关键词搜索**
   - 搜索范围：账号、用户名、邮箱
   - 不区分大小写

#### 用户操作
对每个用户可以执行以下操作：

1. **编辑用户**
   - 修改用户名、邮箱、角色、状态
   - 超级管理员不能修改自己的角色

2. **重置密码**
   - 将用户密码重置为 `123456`
   - 用户下次登录时必须修改密码

3. **删除用户**
   - 删除前需要二次确认
   - 超级管理员账号 (000000) 不能被删除
   - 管理员不能删除其他管理员

#### 添加用户
1. 点击"**添加用户**"按钮
2. 填写用户信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 手机号（可选）
   - 角色（管理员/教师/学生）
3. 系统自动生成可用账号
4. 默认密码为 `123456`

### 权限限制说明
- **超级管理员**: 可以管理所有用户，包括其他管理员
- **管理员**: 只能管理教师和学生，不能管理其他管理员
- **教师/学生**: 无用户管理权限

## 常见问题

### 1. 忘记密码怎么办？
- 联系系统管理员重置密码
- 管理员可以将您的密码重置为 `123456`

### 2. 账号被锁定怎么办？
- 连续5次登录失败会导致账号锁定30分钟
- 请等待30分钟后重试
- 或联系管理员解锁

### 3. 如何修改6位数字账号？
- 6位数字账号是系统自动生成的，不可修改
- 如果需要更换账号，请联系管理员创建新账户

### 4. 浏览器兼容性
系统支持以下浏览器：
- **Chrome**: 最新版本及前两个版本
- **Firefox**: 最新版本及前两个版本  
- **Safari**: 最新版本及前两个版本
- **Edge**: 最新版本及前两个版本

### 5. 移动端支持
- 系统完全响应式设计
- 支持 iOS 12+ 和 Android 8+ 
- 触摸友好的按钮尺寸

### 6. 数据安全
- 所有密码都经过 BCrypt 加密存储
- 传输过程使用 HTTPS 加密
- 敏感操作需要二次确认

### 7. 联系支持
如遇问题，请联系：
- **系统管理员**: admin@your-domain.com
- **技术支持**: support@your-domain.com
